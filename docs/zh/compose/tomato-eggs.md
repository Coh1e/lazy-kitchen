# 番茄炒蛋

<span class="status status-proposed">proposed</span> <span class="cuisine">中 · CN</span>

> ✦ 中国家常菜入门第一道；5 分钟全程 hands-on，无需 SKU/Ratio。

## 组合公式

```
食材 + SOP-STIR-FRY-001 = 一餐
```

## 食材清单（2 人份）

| 食材 | 用量 | 备注 |
|---|---|---|
| 番茄 | 2 个 (≈300 g) | 中等大小，去蒂切瓣（每个切 6–8 瓣） |
| 鸡蛋 | 3 个 (≈150 g) | 打散，可加 1 茶匙料酒去腥 |
| 葱花 | 1 把 (≈10 g) | 葱白与葱绿分开（白入锅、绿撒面） |
| 食用油 | 2 大勺 (≈30 ml) | 分两次下，每次 1 大勺 (≈15 ml) |
| 盐 | 1 茶匙 (≈5 g) | 番茄出汁后再下 |
| 糖 | 1 茶匙 (≈4 g) | 平衡番茄酸味 |

## 时间

| 段 | 分钟 |
|---|---|
| 备菜 | 3 |
| **动手** | **5** |
| 闲置 | 0 |
| 总时长 | 8 |

## 风味结构

- **咸 (Salt)**：盐 + 1 茶匙糖平衡酸味
- **油 (Fat)**：食用油 2 大勺 + 蛋黄自带脂肪
- **酸 (Acid)**：番茄天然酸
- **热 (Heat)**：高火快炒，总 5 分钟

## 硬件

✅ MVP 10 件套即可（炒锅 + 主厨刀 + 锅铲）

## 步骤（来自 SOP-STIR-FRY-001）

1. **备菜**：番茄去蒂切瓣；鸡蛋打散加少许盐；葱白葱绿分开切。
2. **炒蛋**：锅烧热下 1 大勺油 (≈15 ml)，转高火，倒蛋液。蛋液起泡未完全凝固即铲出（约 30 秒），盛碗备用。
3. **炒番茄**：原锅再下 1 大勺油 (≈15 ml)，下葱白爆香 5 秒，倒入番茄瓣。中高火翻炒至番茄边缘软化、出红汁（约 2 分钟）。
4. **调味**：下盐 1 茶匙 (≈5 g)、糖 1 茶匙 (≈4 g) 翻匀。
5. **回锅**：倒回炒蛋，大火翻炒 30 秒让蛋裹上番茄汁。
6. **出锅**：撒葱绿，关火出锅。

## 关键要点

- 蛋要"嫩出"，七八成熟就铲出；老了会发柴
- 番茄一定要炒到出汁，否则味道散
- 糖不是为了甜，是为了"圆"番茄的酸
- 想更浓郁可番茄炒前先用刀划十字、热水烫 30 秒去皮（多 1 分钟工序）

## 复用

- SOP-STIR-FRY-001 复用于：宫保鸡丁、麻婆豆腐回炒、青椒肉丝
- 本菜不依赖任何 SKU/Ratio，是验证 MVP 闭环最简单的范例
