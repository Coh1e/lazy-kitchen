# 使用循环 / Workflow

LK 的灵魂不是"看菜谱做饭"，而是这个 **4 阶段循环**。第一天就理解节奏，后续越用越省心。

## 阶段图

```text
┌─────────────────────────────────────────────────────────┐
│                    周期开始                              │
├─────────────────────────────────────────────────────────┤
│  ① 计划期（10 分钟）                                     │
│     浏览 dish 库，按 hands_on_min 升序挑这周想吃的       │
│     输出：菜单清单                                       │
├─────────────────────────────────────────────────────────┤
│  ② 采购期（一次性，1–2 小时）                            │
│     a) 硬件 / 耗材：缺啥按 buying_guide 配齐             │
│     b) 阶段性大批量：按 SKU.ingredients 列表统采少用调料 │
│     c) 按需小量：on_demand_online 食材在生鲜电商下单     │
├─────────────────────────────────────────────────────────┤
│  ③ 制备期（一次性，半天）                                │
│     按 SKU 公式混合干料 → 茶包袋 → 真空袋 → 标签         │
│     湿酱按 WET SKU SOP 做 → 分装真空 → 冷冻              │
│     熟底按 COOKED SKU SOP 做 → 冷冻成块 → 真空           │
├─────────────────────────────────────────────────────────┤
│                    每天循环                              │
├─────────────────────────────────────────────────────────┤
│  ④ 执行期（每餐 15–30 分钟 hands_on）                    │
│     生鲜电商买当天预切肉/菜（CN 市场预处理）             │
│     从冰箱取一个预制 SKU + 配套 SOP                      │
│     按 SOP 步骤组装 → 上桌                               │
└─────────────────────────────────────────────────────────┘
```

## 关键观念三件事

### 1. ②③ 是低频高投入

一次买齐、一次做完，省下后面所有重复劳动。
这是 LK 模块化论点的人话表达。

### 2. ④ 是高频低投入

因为 ②③ 已经做过，每天只需"取 + 加热 + 切肉 + 端"。
这就是"懒"的兑现。

### 3. 采购的三种节奏

| 节奏 | 例子 | 怎么标 |
|---|---|---|
| 常备 | 盐、生抽、米、蒜、葱 | `procurement_mode: bulk_pantry` |
| 阶段大量 | masala、特色酱、整香料 | `procurement_mode: seasonal_buy` |
| 按需小量 | 柠檬汁、香茅、南姜 | `procurement_mode: on_demand_online` |

`add-dish` skill 在生成新菜时会主动询问每个不在 glossary 的食材属于哪种节奏。

## 周期建议

| 厨房状态 | 推荐周期长度 |
|---|---|
| 新建 LK 厨房 | 第 1 周做 ③ 一次，准备 3–4 个 DRY 包 + 1 个 COOKED |
| 稳定运行 | 每 2–4 周做 ③ 一次，补充消耗的包 |
| 旅行回来 | 周日花半天做 ③，整周 ④ 都轻松 |

## 不要走样

- 不要每天都做 ②③（那就回到了"每顿现做"模式）
- 不要 ④ 时还从零切香料（动手时间会爆涨）
- 不要把 ② 拆成"每天网购小量"（那会回到忙碌循环）

## 反 LK 的反例

**反例 1**：周三晚上 7 点开始想"今晚吃啥"，去看菜谱网站，决定做某道复杂菜，9 点才开吃。
LK 反对：周日就该决定本周菜单 + 备好包。

**反例 2**：每次做菜都从整株蒜开始剥皮、整块姜开始切片。
LK 反对：蒜末/姜末可以批量切完冷冻；或用预制 SKU 包里已经处理好的版本。

**反例 3**：同一个香料柜里堆着 50 种小料瓶子，但 80% 一年只用 1–2 次。
LK 反对：少买偏门小料；如果 30 道菜里只有 1 道用，要么省略，要么按需小量网购。
