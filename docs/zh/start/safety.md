# 食品安全 / Food safety

LK 对食安的态度：**说清楚风险，不当无菌室。**

## 关键风险点

### 真空袋 & 真空封袋器

- 必须 **BPA-free 食品级**（详见 [HW-VACUUM-BAGS](#/zh/buy/consumables-guide)）
- 已用包装 **不可二次封**（交叉污染）
- 微波加热 → 不可（高温析出）
- 冷冻 + 沸水加热 → 可（看耐温标）

### 真空冷冻肉类

- 推荐 -18°C 以下
- 风味期 6–12 个月
- 安全期 24 个月+
- 解冻：冷藏过夜 > 流水 > 微波（按推荐排序）
- **化冻后不可再冻**

### 湿酱 (WET SKU)

- 含蒜 / 含鲜葱的湿酱：**必须冷冻**，冷藏不超过 3 天
- 高酸湿酱（vinaigrette / nuoc cham）冷藏可 2 周
- 无水油基湿酱（chimichurri）冷藏 2 周

### 熟底 (COOKED SKU)

- 先彻底冷却（≤ 2 小时降到室温） → 真空 → 冷冻
- 不要直接热包入袋（凝结水珠加速变质）
- 一次解冻只用一份；解冻量不要超过 1 周用量

### 湿热环境（厨房）

- 砧板：**生熟分离**（推荐 2 块，颜色区分）
- 切完生肉的刀必须立刻洗
- 抹布每天换新（霉菌温床）

### 香料

- 整香料密封避光，3 年内风味好
- 粉香料 6–12 个月开始走味
- 受潮结块的粉香料：扔掉（霉菌风险高）

```

2026-04-25

use by 2027-04-25
```

详见 SOP-LABELING-001（在 add-dish 时会被引用）。

## 出问题怎么办

- 真空包鼓气 → 立即扔（厌氧菌）
- 异味 / 变色 → 不冒险，扔
- 拿不准 → 扔比试更便宜

LK 不替你承担食安责任。所有数据带"建议"性质，落实你自己看判断。
