# 极简硬件 / Minimal hardware

LK 锁定的 **10 件套 MVP**：

```text
刀具:    主厨刀 · 中式菜刀
锅具:    炒锅 · 汤锅
量具:    量勺 · 厨房秤
耗材:    真空封袋器 · 真空袋 · 茶包袋 · 冷冻标签
```

**就这些。**

需要烤箱 / 电饭煲 / 空气炸锅 / 料理机的菜，会在 dish 页顶部明确标 `hardware_extension`，并在 [hardware-extensions.yaml](../../../data/hardware-extensions.yaml) 列 workaround。

每件硬件的完整 buying_guide（选购要点 / 规格建议 / 食安风险 / 常见坑 / 价位）见 [硬件采购指南](#/zh/buy/hardware-guide)。
