# 分装指南 / Packaging guide

## 什么必须分装

- 整香料组合（DRY 包）→ 茶包袋 + 真空
- 干粉组合 → 小自封袋 + 外层真空
- 湿酱 → 真空袋（液体模式封）+ 冷冻
- 熟底 → 冷却后真空 + 冷冻
- 腌肉包 → 肉 + 腌料 + 真空冷冻

## 什么不要分装

| 类型 | 原因 |
|---|---|
| 高频液体调料（生抽、鱼露、米醋） | 消耗快，瓶装直接用 |
| 高频酱料（味噌、豆瓣酱） | 用量不固定 |
| 基础食材（鸡蛋、豆腐、洋葱） | 留人参与感 |
| 生鲜香草（香菜、薄荷、青柠） | 易坏，按需买 |

## 谨慎分装

| 类型 | 原因 |
|---|---|
| 含鲜蒜的湿酱 | 冷藏不宜久放，必须冷冻 |
| 葱油 / 蒜油 | 不建议常温长期存 |
| 液体汤包 | 先冷冻再真空，否则吸入封口机 |
| 干粉直接真空 | 粉末吸入机器，必须先内包茶包袋 |
