# 告示栏 / Bulletin Board

这里是 LK 社区共建的中心。所有新菜、所有讨论、所有迭代都在这里发生。

## 现状

- 看 [proposed](#/zh/board/proposed) — 当前 maintainer 在试验、邀请反馈的菜
- 看 [recent](#/zh/board/recent) — 最近 30 天升级到 approved 的菜
- 看 [history](#/zh/board/history) — 历史所有 status 流转记录
- 学 [contributing](#/zh/board/contributing) — 怎么参与（投票、评论、Issue、PR）

## 状态机

```text
planned (路线图占位)
   ↓ maintainer 跑 /add-dish 起草
proposed (告示栏公开，社区评议)
   ↓ 共识 + 跨 CLI 审通过
review (准备最终审)
   ↓ maintainer 最终拍板
approved (正式发布)
   ↓ 被新版替代
deprecated
```

每个状态对应不同颜色徽章在 dish 页顶部显示。

## 你可以做的事

- 看到感兴趣的 proposed 菜 → 点进去 → 在 Giscus 投票或评论
- 看到 approved 菜也可继续评论 / 提改进建议
- 想加一道完全没人提过的菜 → 提 Issue 求 maintainer 安排，或自己 fork + 跑 skill + 提 PR
