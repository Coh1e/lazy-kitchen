# 断生判断 / Doneness

"什么时候熟了？" 是真正难传的隐性知识。LK 给一份速查表。

## 内温法（最准）

需要 [HW-INSTANT-THERMOMETER](#/zh/buy/hardware-guide)。

| 食材 | 内温 | 状态 |
|---|---|---|
| 牛排 (rare) | 49–54°C | 中心鲜红 |
| 牛排 (medium) | 60–63°C | 中心粉红 |
| 牛排 (well done) | 71°C+ | 全灰 |
| 鸡肉 (安全线) | 74°C | 不可低于 |
| 猪肉 (安全线) | 63°C + 3min 静置 | |
| 鱼 (大部分) | 52–55°C | |
| 三文鱼 (medium) | 49–52°C | |

## 视觉法

- **鸡肉**: 切开看汁色 — 透明 = 熟，粉红 = 没熟
- **虾**: 弯成 C 形 = 熟，O 形 = 老
- **鱼**: 用筷子戳 — 自然分层（layered flake）= 熟
- **意面**: 切开横截面 — 中心无白点 = al dente
- **米饭**: 锅边干 + 中心松 = 熟

## 时间法（不准但够用）

- 鸡腿肉炖 30+ min 一定熟
- 牛腩炖 1.5–2.5 小时 = 软烂
- 米饭电饭煲跳到保温即可

## 闻 / 听

- 油炸：声音从大滋滋转到细密 = 内部水蒸气出尽 = 接近熟
- 烤肉：油脂滴下烟气显著 = 表层焦化中

## 不靠谱的方法

- ❌ "XX 分钟" 死记 — 锅温、肉块大小都影响
- ❌ "颜色" 只看外面 — 内里可能没熟
- ❌ 红外测温枪测肉 — 只测表面

## 安全底线

- 鸡肉、整禽、绞肉、鱼贝若有怀疑 → 多煮 1 分钟比生冒险好
- 不靠"我相信我的眼睛" 替代温度计
