# 备菜纪律 / Mise en place

> Escoffier 的厨房纪律：**所有材料就位后再开火**。

## 为什么

- 中餐快炒尤其严格 — 锅一热下去，再切就晚了
- 减少操作中"哎呀忘了..." 的崩溃时刻
- 让 hands_on_min 真的接近 yaml 标的数

## 做法

### 开火前

1. 看完整个 SOP
2. 列所有需要的食材 + 调料 + SKU 包
3. 全部切配 / 量好 / 装小碟摆好
4. 准备好成品装载容器（盘子或碗）
5. 抹布、隔热垫、铲子在伸手可及处

### 装碟方式

```text
小碟 1: 蒜末 + 姜末 + 葱花 (一锅入)
小碟 2: 肉片 + 腌料 (单独)
小碟 3: 蔬菜（按下锅顺序排）
小碟 4: 调味（生抽 + 醋 + 糖，先量好）
小碟 5: 收尾（淀粉水 + 香油，临出锅用）
```

### 然后才开火

按 SOP 步骤推进。不打开微信，不接电话。

## 中式 SKU 的好处

如果用了 LK 的 DRY/WET 包：
- DRY 包就是你的 "调料碟" 缩成一袋
- WET 包就是你的 "腌料碟" 缩成一份
- 切配（CUT-*）大部分可由生鲜电商代劳

→ mise en place 时间可压缩到 5 分钟。
