# 耗材采购指南 / Consumables buying guide

LK 用到的非器具耗材，全部带食安考量:

## HW-VACUUM-BAGS / 真空袋

详见 [data/hardware.yaml](../../../data/hardware.yaml) HW-VACUUM-BAGS 条目。

**关键**:
- 厚度 90 微米起步（90/100/120 三档主流）
- 单面光滑+单面纹路（"channel/网纹"）才配家用机；纯光滑只能配腔体机
- **必须 BPA-free 食品级**
- 不可微波；冷冻+沸水加热看耐温标
- **已用包装不可二次封**（交叉污染）

价位：一卷 25×500cm，30–80 元。

## HW-SPICE-SACHETS / 茶包袋

- 食品级无纺布（PP / 玉米纤维），耐沸水
- 抽绳款比订口好用
- 6×8cm + 8×12cm 各备一包
- **PE 塑料茶包袋遇高温析出 → 勿用**
- 真空封外袋之前先把香料装入茶包袋（避免颗粒被吸入封口机）

## HW-FREEZER-LABELS / 冷冻标签

- 必须冷冻防水（普通快递贴纸冻 1 周脱胶）
- 配油性笔（zebra mckee / sharpie）
- 双面标签纸（前撕后揭）方便贴
