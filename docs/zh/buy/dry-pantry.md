# 干货母库 / Dry pantry

详见 [data/dry-pantry.yaml](../../../data/dry-pantry.yaml)。

这一层是 LK 的策略核心 —— **大批量采购干香料 + 按 SKU 公式分装**。

## 整香料

八角、香叶、桂皮、白胡椒粒、花椒、陈皮、干辣椒

## 粉香料

孜然粉、甜椒粉、烟熏甜椒粉、辣椒粉、咖喱粉、姜黄粉、蒜粉、洋葱粉、黑胡椒粉、白胡椒粉

## 干鲜

出汁粉、干香菇、海带芽、海带

## 关键

- 整香料密封避光 → 风味期 1–3 年
- 粉香料 6–12 个月开始走味
- 大批量 + 合理周期使用 = 单价低 + 风味好

## 工作流

阶段 ② 采购期：按你这个月想做的菜，一次性补 dry-pantry 缺口。
阶段 ③ 制备期：按 DRY SKU 公式（如 [DRY-CANTON-CLEAR-BEEF-v1](#/zh/pack/dry/canton-clear-beef)）混合 → 茶包袋 → 真空。
