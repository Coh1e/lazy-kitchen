# 基础食材 / Base ingredients

详见数据真源 [data/ingredients.yaml](../../../data/ingredients.yaml)。

## 关键观念

- 蛋白质和蔬菜不预制（保留人参与感）
- 中国电商可买预切：每个条目带 `cn_market_pre_prepped[]`
- 按 `procurement_mode` 区分采购节奏

## 速览（核心 ~20 项）

### 蛋白质
鸡蛋、豆腐、鸡腿肉、五花肉、牛腩、猪肉片、牛肉片、虾仁、鱼片

### 蔬菜
洋葱、白萝卜、卷心菜、青菜、蘑菇、胡萝卜、土豆、番茄

### 主食
米、面条、河粉、乌冬、意面

完整字段（中英名 / aliases / 储藏 / 采购模式 / 中国预切 / buying_notes）见 yaml。
