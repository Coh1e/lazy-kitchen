# Consumables buying guide

Non-tool consumables LK uses, all with food-safety considerations.

## HW-VACUUM-BAGS

See [data/hardware.yaml](../../../data/hardware.yaml) HW-VACUUM-BAGS entry.

**Key**:
- Thickness 90μ minimum (90/100/120 are the common tiers)
- One smooth + one channeled face required for home sealers; pure smooth only for chamber sealers
- **Must be BPA-free, food-grade**
- Cannot microwave; freeze + boil OK (check rating)
- **Used bags cannot be re-sealed** (cross-contamination)

Price: roll 25×500cm, 30–80 CNY.

## HW-SPICE-SACHETS

- Food-grade non-woven (PP / corn fiber), boil-safe
- Drawstring beats sewn closure
- 6×8cm + 8×12cm both useful
- **PE plastic tea bags leach at high temp → DO NOT USE**
- Bag spices into sachets BEFORE vacuum sealing the outer bag (else granules get sucked into the sealer)

## HW-FREEZER-LABELS

- Must be waterproof freezer-grade (regular shipping labels peel after a week frozen)
- Pair with permanent marker (zebra mckee / sharpie)
- Double-back labels (peel + stick) easier to apply
