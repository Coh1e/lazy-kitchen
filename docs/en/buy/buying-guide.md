# Buying guide overview

Three procurement rhythms:

## 1. Always-stocked (`bulk_pantry`) — large bottles/bags

Examples: salt, light soy, rice, garlic, onion, whole spices.
~20 core items, long-term stocked.

## 2. Periodic bulk (`seasonal_buy`) — moderate batches

Examples: garam masala, specialty pastes, cassia bark.
Restock when opening a new cuisine.

## 3. On-demand small (`on_demand_online`) — order as needed

Examples: bottled lime juice, lemongrass, galangal.
Don't stockpile; buy what a dish needs.

## Why hardware can be minimal

Chinese fresh e-commerce sells pre-cut meat, deboned thighs, shredded radish.
Each ingredient has `cn_market_pre_prepped[]` listing the common pre-prepped forms.
Buy these to skip CUT-* SOPs entirely (they're marked `optional_if_pre_prepped: true`).
