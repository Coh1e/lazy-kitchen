# Doneness

"How do I know it's cooked?" is genuinely hard tacit knowledge. LK gives a quick reference.

## Internal temp (most accurate)

Requires [HW-INSTANT-THERMOMETER](#/en/buy/hardware-guide).

| Item | Internal °C | State |
|---|---|---|
| Steak (rare) | 49–54 | Bright red center |
| Steak (medium) | 60–63 | Pink center |
| Steak (well-done) | 71+ | Fully gray |
| Chicken (safety) | 74 | Mandatory minimum |
| Pork (safety) | 63 + 3min rest | |
| Fish (most) | 52–55 | |
| Salmon (medium) | 49–52 | |

## Visual

- **Chicken**: cut and check juices — clear = done, pink = not done
- **Shrimp**: C-shape = done, O-shape = overcooked
- **Fish**: poke with chopstick — natural flakes = done
- **Pasta**: cross-section — no white center = al dente
- **Rice**: pot edge dry + center fluffy = done

## Time-based (imprecise but workable)

- Chicken thigh stewed 30+ min = definitely cooked
- Beef brisket 1.5–2.5 hr = tender
- Rice cooker click to keep-warm = done

## Smell / sound

- Deep fry: large sizzle → fine sizzle = water steamed out = nearly done
- Roast meat: visible smoke from rendered fat = surface searing well

## Unreliable methods

- ❌ Fixed minutes by clock — pan temp + meat size both vary
- ❌ Color alone — outside vs inside differs
- ❌ IR thermometer on meat — measures surface only

## Safety floor

- Chicken, whole birds, ground meat, raw fish: when in doubt, cook 1 more minute (better than risk)
- Don't substitute "I trust my eyes" for an actual thermometer
