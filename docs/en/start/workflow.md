# Workflow

The soul of LK is not "follow recipes" but this **4-phase loop**.

## Phase diagram

```text
┌────────────────────────────────────────────────────┐
│                  Cycle starts                       │
├────────────────────────────────────────────────────┤
│  ① Plan (10 min)                                    │
│     Browse dish library; pick by hands_on_min asc.  │
│     Output: this week's menu                        │
├────────────────────────────────────────────────────┤
│  ② Buy (one-time, 1–2 hr)                           │
│     a) Hardware/consumables per buying_guide        │
│     b) Bulk-buy seasonal condiments per SKU recipes │
│     c) On-demand small batches via fresh e-commerce │
├────────────────────────────────────────────────────┤
│  ③ Pack (one-time, half-day)                        │
│     Mix DRY combos → spice sachets → vacuum bags    │
│     Make WET sauces → vacuum → freeze               │
│     Cook COOKED bases → freeze in blocks → vacuum   │
├────────────────────────────────────────────────────┤
│                  Daily loop                         │
├────────────────────────────────────────────────────┤
│  ④ Execute (15–30 min hands_on per meal)            │
│     Order pre-cut meat/veg from e-commerce          │
│     Grab a SKU pack + matching SOP                  │
│     Follow SOP → plate → eat                        │
└────────────────────────────────────────────────────┘
```

## Three procurement rhythms

| Rhythm | Examples | Tag |
|---|---|---|
| Always-stocked | Salt, soy sauce, rice, garlic, scallion | `bulk_pantry` |
| Periodic bulk | masala, specialty pastes, whole spices | `seasonal_buy` |
| On-demand small | Lime juice, lemongrass, galangal | `on_demand_online` |

## Why this works

- ②③ are low-frequency, high-investment
- ④ is high-frequency, low-investment because ②③ already done
- The "lazy" payoff is in ④
