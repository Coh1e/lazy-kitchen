# Minimal hardware

LK locks the **10-piece MVP**:

```text
Knives:       Chef's knife · Chinese cleaver
Cookware:     Wok · Soup pot
Measurement:  Measuring spoons · Kitchen scale
Consumables:  Vacuum sealer · Vacuum bags · Spice sachets · Freezer labels
```

**That's all.**

Dishes needing oven / rice cooker / air fryer / blender must declare `hardware_extension` and have a workaround in [hardware-extensions.yaml](../../../data/hardware-extensions.yaml).

Per-item buying guide (key considerations / spec / food safety / pitfalls / price band) → [hardware-guide](#/en/buy/hardware-guide).
