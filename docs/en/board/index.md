# Bulletin Board

Center of LK's community-built model. All new dishes, all discussions, all iterations happen here.

## Now

- See [proposed](#/en/board/proposed) — dishes maintainer is currently testing, inviting feedback
- See [recent](#/en/board/recent) — dishes promoted to approved in the last 30 days
- See [history](#/en/board/history) — all status transitions
- Learn [contributing](#/en/board/contributing) — how to participate (vote, comment, Issue, PR)

## State machine

```text
planned (roadmap placeholder)
   ↓ maintainer runs /add-dish
proposed (public on bulletin board, community votes)
   ↓ consensus + cross-CLI review passes
review (preparing final audit)
   ↓ maintainer final approval
approved (officially published)
   ↓ replaced by new version
deprecated
```

Each status maps to a colored badge on the dish page.

## What you can do

- See an interesting proposed dish → click in → vote/comment via Giscus
- Approved dishes also stay open to comment / improvement suggestion
- Want a dish nobody proposed → file an Issue asking maintainer to schedule, or fork + run skill + submit PR
