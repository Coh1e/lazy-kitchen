# Persona avatars

256×256 PNG, transparent background, used in chef-bot PR body header.

## Required files

- `tangniu.png` — 中餐大厨唐牛
- `remy.png` — 西餐 + 全球兜底大厨 Remy / 雷米
- (新 persona merge 时一并加 `<persona-name>.png`)

## 推荐生成方式

ChatGPT (DALL-E) 或同类工具，按以下 prompt 生 → 透明背景导出 → 直接放本目录。

### 唐牛 (tangniu.png) prompt

```
A cartoon mascot of a Chinese chef in his 30s, square jaw, confident
smirk, wearing a white traditional Chinese chef's tunic with red
trim, holding a wok spatula. Simple flat illustration, 2-tone style
(charcoal #2C2E33 + yolk yellow #F4C430 accents), transparent
background. Inspired by 1990s Hong Kong street-food chef aesthetic.
Original character — must NOT resemble any specific real actor or
copyrighted film character. 1:1 square, vector style, suitable for
GitHub avatar (256x256).
```

### Remy / 雷米 (remy.png) prompt

选项 A — 保留"老鼠厨师"梗 + 完全原创视觉:

```
A cute brown field mouse (NOT grey-blue, NOT large-eared) wearing a
classic French chef's toque (white tall hat) and a small white apron
embroidered with a fleur-de-lis. Standing on hind legs, holding a
tiny copper saucepan. Realistic mouse proportions (small ears,
pointed nose, slim body) — explicitly NOT in Pixar/Disney
3D-cartoon style. Use a flat illustrated 2D look, line-art with
warm color fills (brown fur, cream apron, gold pan). Original
character, no resemblance to any animated film. Square 1:1,
transparent background, 256x256.
```

选项 B — 换成法国公鸡 (零 IP 风险):

```
A proud French Gallic rooster (coq gaulois) mascot, standing in chef's
attire: white toque, red neckerchief, blue-and-white striped apron
(French tricolor). Holding a wooden spoon. Flat 2D cartoon
illustration, bold geometric shapes, friendly expression. Original
character, suitable as project mascot. Square 1:1, transparent
background, 256x256.
```

## 法律 note

All avatars must be **original designs**. Do not reproduce specific
copyrighted character likenesses (Disney/Pixar/film studios). Use
generic motifs (Chinese chef + 90s HK street vibe; cartoon mouse + French
toque; etc.) without literal lifts.
